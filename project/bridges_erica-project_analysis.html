Fat Wreck Chords | Project Analysis

  • Project: Re-Design
  • Web Designer: Erica Bridges

Site Intent

The primary focus of this re-deign is to eliminate visual confusion and make the site more user friendly and aesthetically pleasing. As it stands, the site applies harsh color combinations and clutter. It is very difficult to identify the logo as it clashes with the picture background of a band member. We also need to get this site on the list! Currently, the website and has barely had any viewers in the last 3 months.

Site re-design will be accomplished by applying a navigation that runs across the top of the page, as opposed to the current site that applies a break between each link. Also, by combining the media content (ie. Videos, Records, MP3s & Pics) into one drop down menu, it will make for a more organized navigation.

Competitive Analysis

Based on the research that was preformed, the site Fat Wreck Chords, has very few visitors. Fat Wreck Chords is an independent record label for punk bands. According to Alexa.com, Fat Wreck Chordsis ranked at 8,446,309. The site itself has been online for fourteen years and in the last 3 months, has not had any viewers. The target audience for this site includes males and females, from the age of 18-34 yrs.
